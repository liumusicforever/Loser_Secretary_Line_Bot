# 魯蛇秘書
## 源由：
現代的科技人常常有單身寂寞覺得冷的生活，而科技正是解決人類問題的萬靈丹，我們研發了魯蛇秘書，來陪伴這些單身的朋友。
## 宗旨：
解決單身的人的各種心理、生理上的需求
 

## 指令與說明：
## 療育心靈
### [未婚妻]
		秘書會為您送上一個帥哥，僅供療癒身心，釋放壓力。
### [未婚夫]
		秘書會為您送上一個美女，僅供欣賞，止渴用。
## 解決生理需求
### [我愛[關鍵字]]
		秘書會依據主人的需求，推薦最佳的成人電影，以供解決生理需求。 
### [有錢是老大]
		秘書會為主人挑選適當的一日伴侶，陪伴孤獨的主人
