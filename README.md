# 魯蛇秘書
## 源由：
現代的科技人常常有單身寂寞覺得冷的生活，而科技正是解決人類問題的萬靈丹，我們研發了魯蛇秘書，來陪伴這些單身的朋友。
## 宗旨：
解決單身的人的各種心理、生理上的需求
